# AAS Startup Master Checklist

Use this as the weekly working checklist behind the dashboard. Do not put private personal data in this file.

## 1. Privacy-first setup
- [ ] Decide what address will appear on public records.
- [ ] Select registered agent.
- [ ] Select digital office / CMRA mailbox / mail scanning provider.
- [ ] Confirm whether the provider address can be used as the principal office street address where required.
- [ ] Create public business email for state filings and vendor accounts.
- [ ] Create business phone/VoIP number.
- [ ] Create password manager vault and recovery plan.
- [ ] Document who checks mail, how often, and how urgent mail is escalated.

## 2. Florida LLC formation
- [ ] Search Sunbiz name availability.
- [ ] Confirm exact legal name.
- [ ] File Articles of Organization only after address/privacy plan is set.
- [ ] Download filed Articles and document number.
- [ ] Draft operating agreement.
- [ ] Save annual report deadline reminder: May 1 every year.
- [ ] Save registered agent renewal reminder.

## 3. Tax, EIN, banking, accounting
- [ ] Apply for EIN free directly through IRS after LLC exists.
- [ ] Save EIN confirmation letter.
- [ ] Open business checking.
- [ ] Open tax reserve savings.
- [ ] Connect bookkeeping.
- [ ] Set receipt capture workflow.
- [ ] Confirm CPA/tax classification plan.
- [ ] Confirm local/county business tax receipt or license needs.

## 4. Operating stack
- [ ] Business email workspace.
- [ ] Calendar booking page.
- [ ] Cloud drive folder structure.
- [ ] E-signature tool.
- [ ] CRM/pipeline tracker.
- [ ] Invoice/payment workflow.
- [ ] Website form sends to a real destination.
- [ ] Backup/export routine.

## 5. Working templates
- [ ] Proposal template.
- [ ] Statement of Work template.
- [ ] Service terms / MSA placeholder for attorney review.
- [ ] Invoice template.
- [ ] W-9 request packet.
- [ ] Discovery questionnaire.
- [ ] Kickoff agenda.
- [ ] Meeting notes template.
- [ ] Follow-up sequence.
- [ ] Capability statement.

## 6. Market presence
- [ ] LinkedIn company page.
- [ ] Founder LinkedIn headline/about updated.
- [ ] X/Twitter handle reserved if useful.
- [ ] YouTube handle reserved if useful.
- [ ] GitHub/org handle reserved if useful.
- [ ] Social bios match website positioning.
- [ ] Branded email signature.
- [ ] First 4 content topics drafted.

## 7. Federal / SDVOSB path
- [ ] Login.gov account.
- [ ] SAM.gov entity registration data staged.
- [ ] NAICS/PSC list selected.
- [ ] Bank EFT info ready for SAM.
- [ ] SAM submitted.
- [ ] UEI/CAGE recorded when active.
- [ ] DSBS profile drafted.
- [ ] VetCert packet self-check complete.
- [ ] VetCert submitted through MySBA Certifications.
- [ ] APEX Accelerator cadence scheduled.

## 8. Compliance calendar
- [ ] Florida annual report due May 1.
- [ ] SAM renewal every 365 days.
- [ ] Insurance renewal dates.
- [ ] Registered agent / mailbox renewal dates.
- [ ] Federal/state tax deadlines.
- [ ] Quarterly records review.
- [ ] Password/access audit.
